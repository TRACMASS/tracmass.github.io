TRACMASS

Lagrangian trajectory code

Code Reference Gallery Projects Papers

About

  • TRACMASS calculates Lagrangian trajectories from GCM fields
  • Written in Fortran 90/95 standard
  • Used for simulating trajectories on fields from ocean models such as NEMO, MITgcm, and ROMS, as well as atmospheric models such as IFS/OpenIFS from ECMWF
  • Code and installation instructions are available from our GitHub page
